# Limitations

This repo is a recovery and modernization project, not a complete reproduction
of the original private work.

- The original private dataset notebook is unavailable.
- The original 3D model or physical figurine is unavailable.
- The original training environment and full checkpoint provenance are
  incomplete.
- The reconstructed dataset matches the expected data shape, but it is not a
  byte-for-byte recreation of the original dataset.
- Product images helped identify the likely model, but are not redistributed as
  source assets unless licensing is confirmed.

The accurate portfolio claim is:

> Recovered and modernized an old synthetic-data object-detection project from
> partial artifacts, rebuilt its dataset generation path, validated the output,
> and preserved a historical browser demo.
