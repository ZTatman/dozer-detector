# Historical TensorFlow Artifact

`frozen_inference_graph_100k.pb` is preserved from the recovered 2021
EfficientDetJS handoff folder. The modern browser demo uses the exported TFJS
graph model in `demo/tfjs/zach_model`; this frozen graph is kept only as
historical provenance for the original TensorFlow Object Detection API model.

The original training notebook, exact dataset notebook, and checkpoint lineage
were not recovered.
